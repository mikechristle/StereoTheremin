# StereoTheremin
II'm sure you are familiar with the Theremin. 
It is a musical instrument that uses hand movements to control pitch and volume. 
One hand controls pitch, and the other hand control volume. 
I like the expressiveness of a no contact instrument,
 but controlling two variables is just too limiting. 
 With my version each hand controls both pitch and volume. 
 The up/down motion controls volume, and the right/left motion controls pitch. 
 This way the right hand plays treble and the left hand plays base, like a piano.
